FRED Signpost

Federal Reserve Bank of St. Louis — 800,000+ US & international economic time series. No deep map needed: official docs are complete, API is clear, every series is self-annotated.

No auth for CSV Self-documented API 800K+ series Signpost only

Why no deep map?

FRED's official documentation is comprehensive. Every API endpoint is well-documented, every series carries its own title, units, frequency, and seasonal adjustment notes. There are no hidden interfaces to reverse-engineer. A signpost pointing you to the right URL is all you need.

Quick Start — CSV (No Auth)
# Fetch 10-Year Treasury Yield as CSV — no API key needed curl 'https://fred.stlouisfed.org/graph/fredgraph.csv?id=DGS10&cosd=2025-01-01&coed=2026-03-14' # Pattern # https://fred.stlouisfed.org/graph/fredgraph.csv?id={SERIES_ID}&cosd={START}&coed={END} # cosd = start date (YYYY-MM-DD), coed = end date (YYYY-MM-DD)
