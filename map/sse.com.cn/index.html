统一的 thead/tbody 数据表格容器。数据通过 JS 注入，DOM 稳定后可直接 querySelectorAll('tr') 提取行数据。

Bond Overview Controller
债券成交概况页面的数据控制器对象。调用 overviewDay.setOverviewDayParams() 可触发 AJAX 刷新当前日期数据。

Python Quick-Start
# 获取最新债券日度成交概况 import requests, datetime today = datetime.date.today().strftime("%Y-%m-%d") url = f"https://www.sse.com.cn/js/common/sseBond498Fixed.js?searchDate={today}" resp = requests.get(url, timeout=10) # 响应是 JS 赋值语句，提取 JSON 部分 import re, json m = re.search(r'\[.*\]', resp.text, re.S) data = json.loads(m.group()) if m else [] for row in data: print(row) # {'债券品种': '国债', '成交笔数': ..., ...}
